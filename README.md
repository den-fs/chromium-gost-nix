# Chromium GOST для NixOS (Изолированный)

Этот репозиторий содержит Nix-выражение для сборки **Chromium GOST** — версии браузера Chromium с поддержкой криптографических алгоритмов ГОСТ.

## Основные изменения в этом форке

- **Изоляция профиля:** В отличие от стандартных сборок, этот пакет принудительно использует папку `~/.config/chromium-gost`. Это предотвращает конфликты (ошибки блокировки БД, порчу расширений), если в системе одновременно установлен обычный Chromium.
- **Свежая версия:** Обновлено до актуальной сборки на базе Chromium 144+.
- **Интеграция кодеков:** Поддержка проприетарных кодеков через `vivaldi-ffmpeg-codecs`.

## Как это работает (Техническая часть)

Сборка использует `makeWrapper` для добавления флагов запуска:

- `--user-data-dir=$HOME/.config/chromium-gost` — отделяет настройки от основного браузера.
- `--ozone-platform-hint=auto` — для корректной работы в Wayland/X11 окружениях.

## Запуск с поддержкой ГОСТ

Для работы с российскими государственными порталами убедитесь, что в вашей системе установлены и настроены библиотеки КриптоПро. Браузер автоматически подхватит системные сертификаты из хранилища.
